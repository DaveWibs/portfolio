AI-Powered Support Co-pilot - Apollo

Project Screenshot

Project Overview

Apollo is a portable, intelligent technical support system leveraging natural language processing to automatically handle technical inquiries. This AI-powered co-pilot assists support teams by providing instant, accurate responses to common technical questions.

Challenge

As a company matures, knowledge gaps appear over time. People move on and leave documentation that gets forgotten. All staff, regardless to length of tenure should be able to assist clients in times of need and regardless of expertise.

Solution

Instead of allowing that documentation to fall through the gaps Apollo can ingest it from any source and become your in house SME. Utilizing the OpenAI API and langchain, I created a system that can ingest data from any source, convert it to a vector database and be capable of answering any technical support question a user may have based on the documentation provided. Extensible in python, Apollo can utilize any additonal tools you may seek to give it, such as custom log readers or code interpreters.

Results

Constructed, tested and deployed in 93 hours, Apollo was succesful in its goals but did not see widespread usage within the company. It now serves as a basis for future projects involving helpdesk and support initiatives.
